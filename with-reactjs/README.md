# Hygraph ⨯ React.js

[Join our Slack](https://slack.hygraph.com)

This example demonstrates how to query from Hygraph with `graphql-request` in React.js.

## How to Use

### Download Manually

```bash
npx degit hygraph/hygraph-examples/with-reactjs with-reactjs
```

Install & Run:

```bash
cd with-react-js
npm install
npm run dev
# or
cd with-react-js
yarn
yarn dev
```

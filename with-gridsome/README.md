# GraphCMS ⨯ Gridsome

This example demonstrates how to query data from GraphCMS with `@gridsome/source-graphql`.

• [Demo](https://graphcms-with-gridsome.now.sh)

## How to Use

### Download Manually

```bash
npx degit graphcms/graphcms-examples/with-gridsome with-gridsome
```

Install & Run:

```bash
cd with-gridsome
npm install
npm run dev
# or
cd with-gridsome
yarn
yarn dev
```
